## FEATURES

  * Listen to music
  * Simple design
  * User-Friendly Interface
  * Fast Performance
  * Automatic scan of media files
  * Browse/ Sort/ Search albums and artists
  * Create playlists by long pressing on music (removed due to issues)
  * Easily search songs by keywords
